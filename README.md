# Seasonal Vector Framework

**Set. Track. Achieve.**

A local-first year planning system built around stable long-term directions (vectors) across life domains, with quarterly actions that adapt every 90 days.

## Concepts

| Term | Meaning |
|------|---------|
| **Domain** | A life area you're actively developing — career, health, sport, finances |
| **Vector** | A year-long direction within a domain. Stays constant. Guides decisions. |
| **Goal + KRs** | What "done" looks like. Key results tell you if you're getting there. |
| **Actions** | Quarterly concrete steps. Change every 90 days as context shifts. |
| **Anchors** | Fixed dates on the timeline. Hard deadlines that shape planning. |

All data is stored in `localStorage` under two keys:
- `svf_data_v2` — domains, vectors, actions, anchors, calendar notes
- `svf_reviews_v1` — weekly review history

No accounts, no cloud, no tracking. Export JSON from Settings regularly as a backup.
